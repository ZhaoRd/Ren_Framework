AppDomainTypeFinder Classzhaord
A class that finds types needed by SmartStore by looping assemblies in the currently executing AppDomain. Only assemblies whose names matches certain patterns are investigated and an optional list of assemblies referenced by CustomAssemblyNames are always investigated.
