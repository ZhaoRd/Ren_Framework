The disposable sync lock
Remarks
This method internally wraps either a ReaderWriterLockSlim or an empty noop action dependending on the scope of the underlying ICache implementation. The static (singleton) cache always returns the ReaderWriterLockSlim instance which is used to sync read/write access to cache items. This method is useful if you want to modify a cache item's value, thus must lock access to the cache during the update.
